# Presentation Timeline Builder

A simple single-file web app for building presentation-friendly timelines with:

- start date
- duration
- person responsible
- notes
- clean timeline view for presenting or printing
- JSON export/import for sharing underlying timeline data

## GitHub Pages hosting

1. Create a new GitHub repository.
2. Upload `index.html` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` and `/root`
5. Save.
6. GitHub will publish your site at a URL like:
   `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## How to use

- Add timeline items from the left panel.
- Use **Export JSON** to save your timeline data.
- Use **Import JSON** to reload a saved timeline.
- Use **Print / Save PDF** for presentation handouts.

## Notes

- The app stores data in browser local storage.
- If you open the hosted app on another device or browser, import the JSON file to load your timeline.
